📤 동기화 내보내기

✅ 방법 1. 퀵쉐어로 텍스트 보내기 (가장 확실)
받는 기기 화면에 내용이 뜨면 [복사]를 누르고 → 메인 기기에서 [가져오기 → 붙여넣기] 하세요.
방법 2. 퀵쉐어로 파일 보내기 (되면 가장 편함)
받는 기기 다운로드 폴더에 파일이 저장되면 → 메인 기기에서 [가져오기 → 파일 선택]으로 불러오세요.
⚙️ PC에서 쓰거나 공유 버튼이 안 뜰 때만: 아래 칸을 길게 눌러 [전체 선택 → 복사]

📥 동기화 가져오기

✅ 방법 1. 붙여넣어서 가져오기 (가장 확실)
퀵쉐어로 받은 내용이 화면에 뜨면 [복사]를 누른 뒤, 아래 칸을 길게 눌러 [붙여넣기] 하세요.
방법 2. 파일에서 가져오기
퀵쉐어로 받은 파일(다운로드 폴더의 점호동기화_*.json)을 선택하세요.
